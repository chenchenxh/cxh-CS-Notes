## 题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

## 题解

```java
public class Solution {
    public int JumpFloorII(int target) {
        if (target == 1) return 1;
        else if (target == 2) return 2;
        else if (target <= 0) return 0;
        int sum = 0;
        for(int i = 1; i < target; i++){
            sum += JumpFloorII(i);
        }
        return sum+1;
    }
}
```
